## 📖 About The Project

**Ali Store** is a modern, fully responsive ecommerce website that provides a seamless online shopping experience. The website showcases a variety of product categories including Electronics, Mobiles, and Appliances, featuring a clean and professional design that works perfectly across all devices.

This project was built by following and applying the design from the amazing YouTube tutorial series by **Reda Tech (رضا تك)**, with additional custom features and enhancements added on top.

### ✨ Key Features

| Feature | Description |
|---------|-------------|
| 🏠 **Home Page** | Hero slider, promotional banners, and categorized product sections |
| 🛒 **Shopping Cart** | Slide-in cart sidebar with add/remove, quantity control, and subtotal |
| ❤️ **Wishlist (Favourites)** | Dedicated wishlist sidebar with add-to-cart and remove functionality |
| 💾 **Local Storage** | Cart and Wishlist data persists across page refreshes |
| 📱 **Fully Responsive** | Optimized for desktop, tablet, and mobile devices |
| 🍔 **Mobile Menu** | Slide-in navigation for smaller screens |
| 🔍 **Search Bar** | Search input with category filter dropdown |
| ⬆️ **Scroll to Top** | Smooth scroll-to-top button appears on scroll |
| 🎠 **Product Swipers** | Swiper.js carousels for browsing products by category |

---

## 🖥️ Tech Stack

- **HTML5** — Semantic structure & SEO-friendly markup
- **CSS3** — Custom properties, Flexbox, Grid, animations & transitions
- **JavaScript (Vanilla)** — DOM manipulation, event delegation, localStorage API
- **Swiper.js** — Touch-friendly product carousels
- **Font Awesome** — Icons library
- **Google Fonts** — Inter, DM Sans, Cairo typography

## 📂 Project Structure

```
📦 Ecommerce Website
├── 📄 index.html            # Main page
├── 📄 products.json          # Product data (25 products)
├── 📁 css/
│   ├── style.css             # Main stylesheet
│   ├── responsive.css        # Media queries
│   ├── normalize.css         # CSS reset
│   └── all.min.css           # Font Awesome
├── 📁 js/
│   ├── main.js               # Core logic (cart, favourites, UI)
│   ├── items-home.js          # Product rendering
│   └── swiper.js             # Swiper initialization
├── 📁 img/                   # Banners, products, logos
└── 📁 webfonts/              # Font Awesome font files
```

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/engAli983/Ecommerce-Website.git
   ```

2. **Open the project**
   ```bash
   cd Ecommerce-Website
   ```

3. **Run with Live Server**
   - Open `index.html` with VS Code Live Server extension
   - Or simply double-click `index.html` in your browser

> 💡 No build tools or dependencies needed — it's pure HTML, CSS & JS!
